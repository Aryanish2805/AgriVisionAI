# Methodology

## 1. System Architecture
Our proposed system abandons the traditional, monolithic inference pipeline in favor of a novel **Agentic AI Architecture**. The system leverages domain-specific intelligent agents that coordinate asynchronously:
- **Planning Agent**: Decodes farmer objectives from natural language queries.
- **Context Agents (Weather & Market)**: Fetch and synthesize external variables critical to optimal yield.
- **Prediction Agents (Crop & Fertilizer)**: Execute highly tuned, isolated prediction models.
- **Synthesizer Agents**: Include the Reasoning Agent and Explainability (XAI) Agent, combining raw probabilities with logic rules and SHAP interpretations.
- **Recommendation Agent**: Formats the final, highly personalized, and interpretable response.

## 2. Dataset Collection & Preprocessing
The core experiments utilize a robust dataset (e.g., Crop Recommendation) containing essential NPK and environmental factors:
- **Features**: Nitrogen (N), Phosphorus (P), Potassium (K), Temperature (°C), Humidity (%), pH, and Rainfall (mm).
- **Target Variable**: 22 unique crop labels.
- **Preprocessing Pipeline**: 
  - Standard scaling `StandardScaler()` applied to all continuous features to ensure uniform activation functions.
  - Categorical targets are processed via `LabelEncoder`.
  - Splitting is performed using an 80/20 train/test ratio with stratified distribution mapping.

## 3. Advanced Model Architectures
Traditional baselines are tested against modern neural architectures.
- **Multi-Layer Perceptron (MLP)**: Utilizes a (128, 64, 32) deep network structure with ReLU activations trained via Adam.
- **TabNet**: Implemented via `pytorch-tabnet`, offering sequential attention mechanisms for tabular data, blending the interpretability of tree-based methods and the representation capacity of deep networks.

## 4. Explainable AI Integration (XAI)
To demystify deep learning "black boxes", we integrate SHAP (SHapley Additive exPlanations):
- **KernelExplainer**: Applied across our deep architectures to isolate feature marginal contributions.
- Artifacts directly outputted include Feature Importance plots, SHAP Waterfalls (local interpretability), and Summary Plots (global interpretability).
